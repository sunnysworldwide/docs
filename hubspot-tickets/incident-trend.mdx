# Overview

Use this endpoint to retrieve **incident trend statistics** between a start and end date.  

The response includes the list of dates (“labels”) and a numerical data series representing how many incidents occurred on each day.

\<Info\>

This endpoint belongs to the **Stats** section and requires authentication.

\</Info\>

# Authentication

This endpoint requires a valid access token.

\`\`\`bash

Authorization: Bearer \<your_token_here\>

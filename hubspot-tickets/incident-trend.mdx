# Get Incident Trend Statistics

Use this endpoint to retrieve **incident trend data** between two dates.\
You receive a list of date labels and a matching series of incident counts.

<Info>
  This endpoint belongs to the **Stats** section and requires authentication.
</Info>

---

## Endpoint

```http
GET /stats/incidents-trend/:startDate/:endDate
```

### Example Request

```bash
curl -X GET "https://api.example.com/stats/incidents-trend/2024-01-01/2024-01-31" \
  -H "Authorization: Bearer <your_access_token>"
```

---

## Authentication

Include your access token:

```bash
Authorization: Bearer <your_access_token>
```
